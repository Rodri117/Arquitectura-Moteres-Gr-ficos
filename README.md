UCQ---------Arquitectura de Motores Gráficos

#####Rodrigo Delgado Robles######

//Proyecto 1°Parcial//

Ovjetivo: 
Mover el Cubo
- Arriba
- Abajo
- Izquierda
- Derecha

Escala del Cubo
- Se Aleja 
- Se A cerca

Teclas con las que se puede mover el Cubo

- Arriba = Tecla "W"     - Derecha = Tecla "D"       - Se Aleja = Tecla "I"
- Abajo = Tecla "S"      - Izquierda = Tecla "A"     - Se A cerca = Tecla "O"
